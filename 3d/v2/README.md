# IR-Golf-Ball 3D Model v2

## What's Changed?
- You can assemble it without superglue!
- Easy battery change. The inner weight stays in the core when you take off one of the shells. 
- Uses M5x16mm set screws for counter weight. Easy procurement. 
- Calculated balance on CAD

## How to Assemble 
*Only explains differences from v1. Please follow v1 assembly instructions for the rest

### Counter Weight
Insert two M5x16mm set screws into the 3D-printed counter weight from each side.
![Assemblying Counter Weight](<img/counter_weight.png>)


### Inner Weight
Place 4mm metal balls on the 3D-printed inner weight and gently push them into the holes one by one. Some holes require two balls in a row. Repeat for both inner weights.
![Insert balls into inner weight](<img/inner_weight.png>)
  
Place two 2mm spacers on one of the inner weights and put the other one on top. Fix them with two M2 self-tapping screws. 
![Assembly inner weight](<img/inner_weight_sand.png>)
